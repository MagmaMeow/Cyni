Features

Moderation

Equip your server's moderation team with over 20 powerful commands, including automated moderation features, all customizable via the config command.

Staff Management

Streamline the management of staff pinfractions and application results with the Staff Management feature, which also tracks message activity for efficient oversight.

Leave Of Absence/Quota Exemption

Simply request LOA from Cyni, approve/decline/extent/void/delete staff LOA or RA according to requirements.

Server Logging

Ensure comprehensive server event tracking with Cyni Audit Logging, which captures key actions such as member joins, leaves, message deletions, and edits.

Ban Appeal Module

Implement a streamlined Ban Appeal module to handle user appeals efficiently, allowing your moderation team to review, manage, and resolve appeals with ease.

Utility & Fun Commands

Utilize a variety of utility commands, including 'whois' for user information, and fun commands for displaying random dog and cat images.

Giveaway

Host giveaways with ease using Cyni's giveaway command, which allows you to create and end giveaways with ease.

Cyni Logging

Ensure better security, with cyni command logging & cyni config change log. Which will help servers to prevent abuse of commands.

Easy Config

Customize Cyni to your server's needs with the Cyni Config command, which allows you to change configuration.
